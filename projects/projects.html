Computer Science Projects

Project 1: FCOS Object Detector

Technologies used: Python, Jupyter Notebooks, Computer Vision, Pytorch

Description: Implemented and trained a Fully Convolutional One-Stage Object Detection model using PASCAL VOC 2007 dataset. Scaled image to different resolutions with RegNetX-MF400MF backbone, then created feature maps from each in feature pyramid to detect objects of different sizes

Project 2: Vending Machine

Technologies used: Python, React Native, Node JS, REST APIs, Database, Docker, Microservices, Software Engineering, Git, CI/CD, Embedded Programming

Description: Worked with a team of 6 in Software Development environment to design and implement vending machine from scratch, using Python for inventory management, keeping inventory updated in MYSQL database. Client side communicate via REST API with Javascript Backend Server which handles database communication and Stripe API calls for payment processing. JS server and database hosted on Docker containers with Docker Compose Created physical machine, running Vending Machine code on Raspberry Pi, vendor interface is website built with React Native used to keep track of inventory and restock
